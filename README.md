Finance Budget vs Actual Analysis Dashboard

Project Overview

This project analyzes the differences between budgeted and actual financial expenses using Excel and Power BI. The objective is to compare planned budgets with actual expenditures and identify spending patterns and variances.

Tools Used

- Microsoft Excel (Data Cleaning and Transformation)
- Power BI (Data Visualization and Dashboard Creation)

Dataset Information

- Dataset Name: Budget vs Actual Financial Dataset
- Source: Kaggle
- Dataset Link: https://www.kaggle.com/datasets/kennathalexanderroy/budget-vs-actual-financial-dataset

Files Included

1. Cleaned Dataset (.xlsx)
2. Power BI Dashboard (.pbix)
3. Dashboard Report (.pdf)

Dashboard Features

- KPI Cards for Total Budget, Actual Expense, Variance, and Transactions
- Monthly Expense Trend Analysis
- Budget Status Distribution
- Variance Analysis by Category
- Budget vs Actual Comparison by Department
- Interactive Slicers for Department and Region

Conclusion

The dashboard provides insights into financial performance by comparing budgeted and actual expenses. It helps identify overspending areas and supports better financial planning and decision-making.
